Node Types

Idea
raw insight, captured as it arrived
Pattern
recurring theme, discovered by Distiller
Practice
intentional behavior, crystallized from idea
Decision
choice made, with context preserved
Synthesis
AI-discovered bridge between clusters
Integration
embodied insight, only humans assign this

Edge Types

-> LEADS_TO A develops into B over time
-> GENERATES A directly produced B
-> SURFACES_FROM A became visible because of B
<> CONTRADICTS A and B are in productive tension
<> ASSOCIATED_WITH related, no causal claim
-> INFORMS A provides context for B

Node Schema

id: uuid-v4
type: Idea | Pattern | Practice | Decision | Synthesis | Integration
content: string (user's words, unedited)
energy_level: 1-5 (inferred from language)
created_at: ISO 8601
source: telegram
tags: string[] (AI-generated)

Curator

real-time / every incoming message
Message arrives. Curator checks complexity.
Simple thought: classify type, generate tags, infer energy from language.
Complex thought: ask 1-2 clarifying questions, then classify.
Question directed at Bodhi: route to Claude, do not file.
Write node to vault. Embed in ChromaDB. Send ✓.

Distiller

cron 0 6 * * * / daily at 6am
Query last 7 days of vault nodes.
Group by tag frequency. Compute energy averages per theme.
Find tags appearing 3+ times: pattern candidates.
Claude Opus synthesizes digest. Send via Telegram.
If < 3 nodes in window: silence. No message sent.

Janitor

cron 0 3 * * 0 / Sunday 3am
Full vault scan. Find orphan nodes (zero edges, older than 7 days).
Cosine similarity on all embeddings. Flag pairs > 0.92.
Detect broken edges (missing node references).
Send hygiene report. Wait for human approval before any action.
Never deletes. Archives to vault/archive/.

Surveyor

cron 0 2 * * 6 / Saturday 2am
Embed all nodes via Ollama nomic-embed-text.
HDBSCAN clustering: min_cluster_size=5, min_samples=3, metric=cosine.
Claude labels each cluster (3-5 words).
Betweenness centrality: find bridge nodes between clusters.
Generate Synthesis nodes with SURFACES_FROM edges.
Run SOC analysis: energy distribution, power law check.
Send weekly insight summary.

Nudge System

triggered by Surveyor SOC analysis
Cluster avg energy_level > 3.5 for 3+ consecutive weeks?
Pattern appeared in 3+ Distiller reports?
One message: "Ready to act?"
No scheduling. No reminders. No guilt. The idea reached its own readiness.

Self-Organized Criticality

Per Bak (1987). Grains of sand accumulate on a pile. When a local threshold is exceeded, a cascade occurs. The system naturally evolves toward a critical state where avalanches of all sizes are possible. No central controller. No schedule. Emergence.

P(s) ~ s^(-tau) where tau ~ 1.5

The distribution of avalanche sizes follows a power law. Small cascades are frequent. Large ones are rare but inevitable. This is the mathematical signature of a system at criticality.

Applied to Cognition

Ideas are grains. The vault is the sandpile. Energy accumulates through recurrence. The same insight surfaces in different contexts. The same problem shows up wearing different clothes. At some threshold, the cognitive cascade occurs: a decision becomes obvious, an action becomes necessary, a pattern becomes undeniable.

cluster_energy = avg(energy_level) across nodes in cluster
cascade_threshold = 3.5 sustained over 3+ weeks

What Surveyor Measures

Power law distribution of energy_level across all vault nodes. If the distribution approaches power law, the vault is at criticality: patterns can emerge at any scale.

Cluster energy averages and week-over-week change. Rising energy in a cluster means accumulation. Sustained high energy means approaching threshold.

Betweenness centrality of individual nodes. Bridge ideas carry the most cascade potential because they connect otherwise-separate clusters.

Spaced Repetition Layer

Ebbinghaus forgetting curve applied to ideas, not flashcards. High-energy nodes that have not been acted on surface at increasing intervals: 1 day, 3 days, 7 days, 21 days, 60 days.

This is not a reminder system. It is a mirror. The thought surfaces because it has energy, not because a calendar says so. If you dismiss it, the interval extends. If you engage, it feeds back into the Distiller.

Minimal Intervention Principle

Bodhi does not advise. Bodhi does not coach. Bodhi does not act as a therapist.

It is a data structure that reflects your own thinking. It notices what you keep returning to. It finds connections you did not see. It tells you when a cluster of ideas has been building energy long enough to act on.

One question at the right time: "Ready to act?"

The human decides. Always.

Network Isolation

Threat: gateway exposed to internet
Gateway binds 127.0.0.1 only. UFW blocks port externally. Startup assertion rejects non-loopback binding. Docker network with egress whitelist (api.anthropic.com only).

Telegram Auth

Threat: unauthorized message injection
User ID whitelist. Only the owner's Telegram ID is accepted. All other messages are silently dropped. No response, no acknowledgment, no error.

Vault Isolation

Threat: other processes reading vault files
chmod 700 on vault directory. Owned by dedicated bodhi OS user. Docker volume mounted with uid/gid. No other process can read vault data.

Token Leakage

Threat: API keys visible in UI or logs
Core patch to OpenClaw: redact token-pattern strings before rendering in WebChat UI. Environment variables never logged. .env in .gitignore.

Message Deduplication

Threat: duplicate message flooding
Circuit breaker with 60-second TTL Map. Same message ID within window is dropped. Prevents duplicate vault writes from Telegram retry behavior.

Data Sovereignty

Threat: thinking data in someone else's cloud
All vault data local filesystem. Anthropic receives message text only for classification. Graph structure, energy levels, tags, edges never leave the machine. No cloud sync. No remote backup unless user configures it.

Remote Access

Threat: SSH brute force or open ports
SSH key-only authentication. Password auth disabled. Tailscale for secure remote access without exposing any ports. UFW denies all except SSH.

Supply Chain

Threat: malicious dependencies
pnpm lockfile pinned. OpenClaw upstream vetted (277k stars, active maintenance). Bodhi skills are the only custom code. Minimal dependency surface.
